CEBSR Query Prototype

  • Help

This prototype shows how ontologies can be used to drive a search interface for the CEBSR API.

  1. A Google Sheet is used to define a number of terms, and import terms from other ontologies
  2. A Web Ontology Language (OWL) file is generated
  3. The OWL file is used to generate the tree views for this prototype
  4. The tree views are used to generate GraphQL query strings
  5. The query strings are executed against the CEBSR API, and a table of results displayed
